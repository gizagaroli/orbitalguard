# OrbitalGuard

OrbitalGuard é uma plataforma para monitoramento de objetos em órbita terrestre, análise de riscos de colisão e visualização de dados espaciais. O projeto foi desenvolvido como uma solução com foco em design, experiência do usuário e conscientização sobre o problema dos detritos espaciais.

## Objetivo

- Visualizar objetos em órbita.
- Identificar possíveis riscos de colisão.
- Exibir alertas em tempo real.
- Apresentar informações sobre diferentes regiões orbitais.
- Tornar o tema mais acessível através de uma interface intuitiva.

## Funcionalidades

### Página Inicial
- Apresentação do problema dos detritos espaciais.
- Estatísticas sobre objetos em órbita.
- Explicação do efeito Kessler.
- Navegação para as demais áreas do sistema.

### Tecnologia
- Fluxo de funcionamento da solução.
- Fonte dos dados

### Mapa de Órbitas
- Visualização interativa das órbitas.
- Filtro por região orbital.
- Simulação de objetos monitorados.

### Alertas
- Lista de alertas de colisão.
- Classificação por nível de risco.
- Filtros de visualização.
